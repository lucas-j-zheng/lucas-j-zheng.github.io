## Overview

Your personal concierge agent—an AI voice assistant that handles restaurant calls for you. Make reservations and check current wait times without any manual phone interaction.

## How It Works

1. **Request** - Tell the agent which restaurant you want and what you need (reservation or wait time)
2. **Call** - The agent places a real phone call to the restaurant
3. **Converse** - AI handles the conversation naturally with restaurant staff
4. **Confirm** - Receive confirmation of your reservation or wait time info

## Technical Stack

- **Voice/Telephony:** Twilio for placing and managing phone calls
- **AI/LLM:** Google Gemini for natural language understanding and generation
- **Backend:** FastAPI for the API server
- **Frontend:** Next.js for the user interface

## Why It Matters

Phone calls for simple tasks like checking wait times or booking a table are tedious. This agent eliminates that friction, acting as your personal concierge for restaurant interactions.
