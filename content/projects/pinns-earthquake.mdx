A deep learning approach to predict earthquake and seismic event source locations from surface sensor readings, combining physics-based constraints with intelligent sampling strategies.

## Approach

- **Physics-Informed Neural Networks (PINNs):** Embed the wave equation directly into the loss function
- **RL-Driven Adaptive Sampling:** Use reinforcement learning to intelligently select collocation points
- **Marmousi2 Dataset:** Validated on the Marmousi2 synthetic geology benchmark

Course project for CSCI 2470 — Deep Learning, Brown University.
